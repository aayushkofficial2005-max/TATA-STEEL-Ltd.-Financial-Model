# TATA-STEEL-Ltd.-Financial-Model
## 📝 Recommended README Structure

A comprehensive financial modeling project analyzing **Tata Steel**, built entirely in Microsoft Excel. This model projects future financial performance and estimates the company's intrinsic value.


---

## 🛠️ Key Features & Components
* **Three-Statement Integrated Model:** Fully linked Income Statement, Balance Sheet, and Cash Flow Statement with dynamic forecasting assumptions.
* **Discounted Cash Flow (DCF) Valuation:** Estimated intrinsic value utilizing Free Cash Flow to Firm (FCFF) and Weighted Average Cost of Capital (WACC) analysis.
* **Scenario & Sensitivity Analysis:** Interactive data tables analyzing how changes in revenue growth and discount rates impact the target stock price.

---

## 📂 File Structure
* `Tata_Steel_FM.xlsx` - The complete, fully functional Excel workbook containing all formulas, data models, and sheets.

---

## 🚀 How to Use the Model
1. Download the `Tata_Steel_FM.xlsx` file from this repository.
2. Open the file in **Microsoft Excel** (desktop version recommended for full macro/data table functionality).
3. Navigate to the `Assumptions` tab to modify key drivers like revenue growth rates or cost margins to see real-time updates across the model.
